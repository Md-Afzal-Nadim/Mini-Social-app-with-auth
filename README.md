# 📸 Image Sharing App – MERN Stack

A full-stack image sharing web application built using **React JS, Node.js, Express.js, and MongoDB**.  
Users can create an account, log in, upload images, add descriptions, and view posts shared by other users.

This project includes authentication, image upload functionality, and a responsive UI for both mobile and desktop devices.

Frontend and backend are deployed on **Render**.

## ✨ Features

✅ User Signup & Login Authentication  
✅ Secure User Registration  
✅ Upload Images  
✅ Add Description with Image  
✅ View All User Posts  
✅ Show Username of Uploader  
✅ MongoDB Data Storage  
✅ Image Upload using ImageKit  
✅ Multer for File Handling  
✅ Responsive Design (Mobile + Desktop)  
✅ Frontend + Backend Deployment on Render
